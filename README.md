# TodoList React Class Component

## [➡️Live Link](https://manish99verma.github.io/todolist/)

## Tech stack

- Vite Build Tool
- React Library

## Run Locally
- ### Clone repo

    ```
    git clone https://github.com/manish99verma/todolist.git
    ```
- ### Install Dependencies
    ```
    npm install
    ```
- ### Run development server
    ```
    npm run dev
    ```
